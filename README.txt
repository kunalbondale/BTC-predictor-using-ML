Welcome, This is my Artificial Intelligence & Machine Learning Project = "BTC / USDT Predictor" Made for IIT Kharagpur Hackathon 2024.

In this project, based on the data sets provided by the Institute, i along with my team - Akatsuki, have successfully trained, designed and deployed a BTC Predictor which is based on the data from Jan 2018 - December 2022.

We have used Supervised Learning : Regression and LSTM algorithm to train our model which has an excellent regression score of 0.999 and R score of 0.997.

The whole project was built on Jupyter Notebook & Google Collab Collectively.

This Repository only includes the final model, and not the data cleaning and pre-processing part of the code, which will bw uploaded shortly.
